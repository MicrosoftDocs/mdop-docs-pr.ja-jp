# MED-V イメージを削除する方法


## <a href="" id="bkmk-deletinganimage"></a>MED-V イメージを削除する方法


**MED-V イメージを削除するには**

1.  削除する画像または画像のバージョンをクリックします。

2.  [**画像**] メニューの [**削除**] をクリックします。

    画像が削除されます。
