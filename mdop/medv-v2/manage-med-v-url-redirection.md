# MED-V の URL リダイレクトの管理


MED-V ワークスペースを展開した後は、さまざまな方法から選択して、MED-V ワークスペースに URL リダイレクション情報を追加したり、削除したりすることができます。

このセクションには、割り当てられた MED-V ワークスペースで URL リダイレクション情報を管理するための情報と手順の両方が含まれています。
