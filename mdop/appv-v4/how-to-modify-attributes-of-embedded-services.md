# 埋め込まれたサービスの属性を変更する方法


アプリケーションをシーケンスした後に保存する前に、次の手順を使用して、シーケンス処理されたアプリケーションパッケージを手動で変更することができます。

**埋め込みサービスを削除するには**

1.  [**仮想サービス**] タブをクリックします。

2.  削除するサービスを右クリックし、[**削除**] を選択します。

3.  [ **Sequencer** ] ダイアログボックスで [**はい]** をクリックします。

**埋め込みサービスのプロパティを変更するには**

1.  [**仮想サービス**] タブをクリックします。

2.  プロパティを変更するサービスを右クリックし、[**プロパティ**] を選択します。

3.  [ **Office Source Engine のプロパティ**] ダイアログボックスで、次の作業を実行します。

    1.  [**表示名**] テキストボックスに、サービスの名前を入力します。

    2.  [**説明**] テキストボックスに、サービスの説明を入力します。

    3.  [**実行可能**ファイルのパス] テキストボックスに、実行可能ファイルへのパスを入力します。

    4.  [**スタートアップの種類**] ボックスの一覧で、サービスの開始方法 (手動または自動) を選択します。

    5.  **[OK]** をクリックします。
