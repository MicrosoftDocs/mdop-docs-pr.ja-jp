# [インストール レポート] ページ (詳細情報)


[**インストールレポート**] ページを使って、シーケンス処理の結果を確認します。 これらのレポートは、シーケンス中に発生する可能性のある一般的な問題を診断するのに役立ちます。

このページには、次の要素が含まれています。

<a href="" id="information-pane"></a>**情報ウィンドウ**  
問題と、シーケンス処理全体に関する追加情報が表示されます。
