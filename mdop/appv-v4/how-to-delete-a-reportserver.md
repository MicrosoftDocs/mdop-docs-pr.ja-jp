# レポートを削除する方法


Application Virtualization Server 管理コンソールから、レポートを削除するプロセスは、レポートの種類に関係なく同じです。

**レポートを削除するには**

1.  [**レポート**] ノードをクリックして、レポートの一覧を展開します。

2.  目的のレポート ([**レポート**] ノードまたは [**結果**] ウィンドウのいずれか) を右クリックし、[**削除**] を選択します。
